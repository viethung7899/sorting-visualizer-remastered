This project is the remastered version of my old [repository](https://github.com/viethung7899/sorting-visualizer).

![demo](assets/screenshot.png)

## Improved features

- The website are built in [React](https://reactjs.org/) with context
- The website are styled by [Bulma](https://bulma.io) library
- Cleaner implementation of merge sort

## Common sorting algorithms animated in this project

- Bubble sort
- Insertion sort
- Selection sort
- Quicksort
- Heap sort
- Merge sort

## Installation

The app is already deployed. You can visit it [here](https://sorting-visualizer-5c3b21.netlify.app).

If you want to reeun this app on your local machine, you can clone this repository and install the dependencies

```bash
$ git clone https://github.com/viethung7899/sorting-visualizer-remastered.git
$ cd sorting-visualizer-remastered
$ npm install
$ npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
The page will reload if you make edits.\
You will also see any lint errors in the console.
